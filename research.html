Machine Learning


Real-time break-down prediction

  • Pre-processed the time series came from factory and extracted the useful information from the noisy data.

  • Established the baseline model based on Kernel-SVM (Kernel Support Vector Machine) and use SFS (Sequential Forward Selection) to select features from the data. The result verifies the existence of signals in the pre-processed data.

  • Constructed an auto-encoder to predict anomalies, in which some embedding layers come fist to encode both categorical and continuous features to the same degree.

  • Gave presentation on the Sinto alliance meeting and finished a proposal draft.

Image Processing


Image denoising using sparse representation

  • Applied PDAS (Primal-Dual Active-Set Algorithm) in K-SVD sparse coding stage and formed a new framework K-SVDp in image denoising.

  • Derived the explicit expression for sparse representation in the denoising problem.

  • Conducted several simulations which proved the efficiency of our framework with the state-of-the-art methods according to PSNR (Peak signal-to-noise ratio), SSIM (Structural Similarity Index), visual comparison and reconstruction time.

High Dimension


High dimensional correlation analysis

  • Introduced wdcor (weighed distance correlation) that can assess the association between genetic markers and voxel-wise imaging data.

  • Implanted Weighted distance correlation and its test function in C, and then built an R package “wdcor” using Rcpp.

  • Evaluated the performance of wdcor compared with dCor in ultra-high-dimensional multivariate dependence analysis and conducted a large-scale analysis of data from the Alzheimer's disease neuroimaging project.
